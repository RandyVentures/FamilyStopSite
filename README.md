# FamilyStop Website

Public GitHub Pages site for FamilyStop.

## Overview

FamilyStop helps families find verified family restrooms, changing tables, nursing rooms, playgrounds, and stroller-friendly stops on every trip — from quick errands to cross-country road trips.

## Stack

- Static HTML/CSS/JavaScript
- GitHub Pages hosting
- Pages URL: `https://randyventures.github.io/FamilyStopSite/`
